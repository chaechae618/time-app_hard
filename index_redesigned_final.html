Step 5 — 모델 평가: 각 지표의 의미를 함께 확인하세요.
• MAE/RMSE: 낮을수록 좋음. RMSE는 큰 오차에 더 민감.
• SMAPE: 0~200% 범위. 10% 이하 우수 / 20% 이하 양호 / 20% 초과 미흡
• MASE: 나이브 모델 대비 성능. 1 미만이면 나이브보다 우수
• TS(Tracking Signal): 편향성 지표. |TS| ≤ 4이면 편향 없음 / |TS| > 4이면 편향 의심
• R²: 설명력. 1에 가까울수록 좋음 (0.8 이상 우수)
모델별 SMAPE 비교
값이 낮을수록 더 안정적인 예측입니다. 나이브 기준선과 비교하세요.
앙상블 가중치 (OOS 기반)
Out-of-sample 성능 기반으로 계산된 각 모델의 앙상블 기여도입니다.
STL 분해 (Seasonal-Trend decomposition using Loess): 시계열을 트렌드, 계절성, 잔차로 분리합니다.
• 트렌드 강도 > 0.5: 뚜렷한 장기 방향성 존재
• 계절성 강도 > 0.5: 규칙적 주기 패턴 존재 → 계절성 모델(ETS, SARIMA) 유효
• 잔차: 트렌드·계절성으로 설명되지 않는 불규칙 변동 (노이즈)

잔차 분석: 예측 오차의 패턴을 분석합니다.
• 평균 잔차 ≈ 0: 편향 없는 예측
• ACF 막대가 CI 안에 있으면: 잔차가 백색잡음 → 좋은 모델
• Ljung-Box Q < 20: 잔차에 자기상관 없음 (백색잡음 통과)
• 왜도 ≈ 0, 첨도 ≈ 0: 정규 분포에 가까운 잔차
